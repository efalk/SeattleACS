This is a utility for updating the code plugs based on the
WWARA repeater database. This is a rather specialized tool, so
you're probably not interested in it.

Download latest WWARA-rptrlist-20260317.csv from WWARA.

If different:
  Rebuild Chirp and RT CSV files
  Rebuild web page


Programs under Tools:

  Wwara2Csv.py	Convert WWARA format to Chirp or RT
  		Use --help for full documentation

  update.py	Download csv file from WWARA and do updates.
